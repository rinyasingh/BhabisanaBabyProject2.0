Web App

Providing an interface for Bhabhisana parents to login, or guests to visit, and view their ECD Content.


youandyourbaby.bhabhisana.org.za

Introduction and Context

Bhabhisana parents had extremely limited space on their phones, confirmed by a staff member who said "they often canâ€™t download a WhatsApp video we send them", which highlighted the purpose of the web app. Further motivation can be noted with how Bhabhisana staff preferred the web based interface opposed to sending their videos over WhatsApp - which was more manual and time-consuming. This web app is built on a WordPress installation within a CloudWays server. All development and functionality is customized through Oxygen Builder codeblocks in PHP, HTML and CSS. This allowed the fine-grain control needed when accessing information within the wp databases. Below is a high level overview of Web App with relation to the implemented system.



The key aims of the web app were as follows:

  • Authenticate Users.
  • Display ECD Content based on Public vs Logged In User Permissions.
  • Display and Filter per Category and Language.
  • Assigned Content only viewable for Assigned Users.
  • Lightweight and Usable.

Methodology

As highlighted within the home page, a co-design based scrum approach was our project management. Sessions conducted with Bhabhisana parents and UCT staff included workshops, focus groups, heuristic evalautions, and user (task) evaluations. A high level overview of our timeline and utilization of co-design sessions (and iterations) can be found here.

Authenticate Users

All users can login via the "User Login" button which allows them to securely log into the WordPress website.

Public vs Logged in User Permissions

All ECD Content posts are displayed based on User permissions which is which posts they have been assigned. If a user is logged in, the system visibility status is altered: Prompts for a user to login are removed, a logout button is added, they are personally addressed by name, and they can see their "Assigned Content" category and posts - which is not available to guests. All assigned content posts are styled in a darker purple hue to ensure parents can easily differentiate between that and the public content. This was achieved through consistent category classification per post, within the code-blocks.

Filtering per Category and Language

Customized filtering of categories and languages works well within the website.

Displaying ECD Content Pages

Each non-public ECD Content post is only displayed to that User if they are assigned it. To ensure integrity and safety of data, a rogue user is therefore not able to hard code a URL and view a specific category or post not assigned to their existing user profile. This integrity of the system was achieved through checking if they were assigned in PHP, and removing the section visibility through JavaScript and CSS.

Results

The website showed great usability within the Bhabhisana community. A fully responsive website across all screen sizes and mobile phones exists. The choice of Oxygen Builder, and custom coding functionality, has resulted in a lean web app. The website has an average page size of 170 KB and loads within 1.3 seconds.
This was supported by 19 unique log-ins from parents within the first 2 weeks of evaluations. After these logins, parents expressed that they were very proud that their efforts in the workshops conducted across the past two years have culminated into something that is tangible and so useful to them.
